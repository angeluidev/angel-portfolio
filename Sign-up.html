Sign-up Form

School Project - April 2020

The Sign-Up form is the second iteration of a school project I initially completed in February 2020 for the Web Programming Module at Singapore Polytechnic - a school requirement I completed individually.

I did a revision to make it into a Sign-Up validation form using JQuery. This is the Github page for this Sign-Up form.

Highlights:

  1. Ensure all fields have input and the Terms and Conditions box is checked before user can submit.
  2. Validates email address format.
  3. Check if the password satisfies the password strength criteria by using RegExp.
  4. Show / Hide 'password' and 're-confirm password' so the user can check their input for both fields.
